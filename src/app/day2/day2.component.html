Directives

At the core, a directive is a function that executes whenever the Angular compiler finds it in the DOM. Angular directives are used to extend the power of the HTML by giving it new syntax.

There are 3 kinds of directives -

  1. Components - Directives with templates
  2. Structural directives—change the DOM layout by adding and removing DOM elements.
  3. Attribute directives—change the appearance or behavior of an element, component, or another directive.

Components

Components are just directives with templates. These form the main class having details of how the component should be processed, instantiated and used at runtime.


Structural Directives

A structure directive basically deals with manipulating the dom elements. Structural directives have a * sign before the directive. For example, *ngIf, *ngFor and *ngSwitch.

*ngIf
<button (click)="show=!show">{{show?'Hide':'Show'}}</button>
<h4 *ngIf="show">Hello</h4>

Attribute Directives

They are used to manipulate the appearance and behaviour of elements.

NgStyle gives you control on individual properties. But if you want to make changes to multiple properties at once, creating a class which bundles those properties and adding the class with NgClass makes more sense.


[ngStyle]
<p [ngStyle]="{'background-color': 'yellow'}">Hello there!</p>

Hello there!


Mapping style property to a variable

<p [ngStyle]="{'background-color': myColor}">Hello there!</p>

Mapping the function to style property

cities: any[] = [{ 'city': 'Delhi', 'country': 'India'}, ....];

getCityColor(city: any) {
  switch (city) {
  case 'India': return 'turquoise';
  case 'USA': return 'red';
  ......
  }
}

<ul>
  <li *ngFor="let city of cities"
      [ngStyle]="{'background-color':getCityColor(city.country)}">
</ul>

[ngClass]

The NgClass directive allows you to set the CSS class dynamically for a DOM element.


<ul>
  <li *ngFor="let city of cities"
       [ngClass]="{
      'text-primary':city.country==='India',
      'text-danger':city.country==='USA',
      'text-warning':city.country==='Australia',
      'text-success':city.country==='UK'
      }">{{city.city}} ({{city.country}})</li>
<ul>
  • {{city.city}} ({{city.country}})

Alternative sytax

<ul>
  <li *ngFor="let city of cities"
      [class.text-primary]="city.country==='India'"
      [class.text-danger]="city.country==='USA'"
      [class.text-warning]="city.country==='Australia'"
      [class.text-success]="city.country==='UK'">
      {{city.city}} ({{city.country}})</li>
<ul>
  • {{city.city}} ({{city.country}})
